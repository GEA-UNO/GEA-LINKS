Este es un tutorial para rellenar el formulario de manera correcta
Tag: este es un nombre por el cual se identifica univocamente al link a crear, es importante que este no se repita, lo apropiado es que tenga una estructura "pagina-nombre", donde pagina es el padre y puede ser alguno de estas:

[ enf, inicio, info, admin, info-comun, redes, links, links-docs, hum, tutos, ingq ]
Link: esta es la url a la cual queres referenciar, debe tener http o https.
Titulo: el texto principal visible en la tarjeta, no hay restricciones.
Descripcion: el texto secundario visible en la tarjeta, es opcional, evitarlo si el titulo es auto-descriptivo.
Pagina padre: indica en que paginas aparecera la tarjeta, puede aparecer en mas de una, la estructura es "/pagina" o "/pag1,/pag2,...,/pagN", deben estar separadas por coma, sin espacios, lista de padres:

Icono como imagen: determina si el icono es a partir de un link a una imagen, por defecto es un vector(svg).
Icono: el icono/imagen a mostrar. El width/font-size puede ser 24px o 22px, ir cambiando y viendo como se ve. El color lo mismo, usar una app externa para elegirlo y el preview para ver ocmo queda. El icono elegirlo aqui , lo buscas, seleccionas y copias los datos de interes del tab 'HTML'

Si es una imagen debe ser: src='url a imagen' loading='lazy' width='24px'.

Sino la siguiente: data-inline='false' data-icon='nombre del ic' style='font-size: 24px; color: #e02f6b;'.
Password: si esta no es valida no se creara el link.
